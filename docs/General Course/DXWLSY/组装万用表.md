# 组装万用表实验数据处理

1.  **改装5mA，10mA量程电流表**

|                   |      |
|-------------------|------|
| 满偏电流$I_{g}$： | 1mA  |
| 内阻r：           | 240Ω |

根据方程组：$\left\{ \begin{array}{r}
(R_{1} + R_{2}) \ast (I_{1} - I_{g}) \\
(R_{2} + r) \ast I_{g} = R_{1} \ast (I_{2} - I_{g})
\end{array} \right.\ $ 有$R_{1} = R_{2} = 30\Omega$

据此进行实验，数据记录与处理：

|              |      |      |      |      |      |
|--------------|------|------|------|------|------|
| I_(准)（mA） | 1    | 2    | 3    | 4    | 4.85 |
| I_(改)（mA） | 1.05 | 2.10 | 3.17 | 4.21 | 5    |
| ΔI（mA）     | 0.05 | 0.10 | 0.17 | 0.21 | 0.15 |

校准曲线如下：
